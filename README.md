# 🏥 Hospital Emergency Room Dashboard | Microsoft Excel

## 📌 Project Overview

This project showcases an interactive **Hospital Emergency Room Dashboard** built using **Microsoft Excel** to analyze emergency department performance and patient data. The dashboard transforms healthcare data into actionable insights through KPIs, charts, and interactive filters, helping stakeholders monitor patient flow, service efficiency, and resource utilization.



## 📂 Project Files

* `Hospital Emergency Room Dashboard.xlsx` – Interactive Excel dashboard
* `Hospital Emergency Room Dataset.xlsx` – Source dataset
* `Images/Dashboard.png` – Dashboard screenshot
* `README.md` – Project documentation

## 📈 Dashboard Features

- Interactive KPI Cards
  - Number of Patients
  - Average Wait Time
  - Patient Satisfaction Score

- Patient Admission Analysis
  - Admitted vs. Not Admitted patients

- Age Group Analysis
  - Patient distribution across different age groups

- Timeliness of Care
  - Percentage of patients attended within 30 minutes

- Gender Distribution
  - Male vs. Female patient analysis

- Department Referral Analysis
  - Referral trends across hospital departments

- Interactive Filters
  - Month
  - Year
